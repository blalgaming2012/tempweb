# وثيقة متطلبات تطبيق الويب\n
## 1. اسم التطبيق\nنظام إدارة الخدمات والعمال

## 2. وصف التطبيق
تطبيق ويب متكامل لإدارة خدمة العملاء والعمال، مبني على حزمة تقنيات حديثة (React/Vite، Tailwind CSS، Supabase) مع تركيز على تجربة المستخدم الأنيقة والأمان العالي.

## 3. الوظائف الرئيسية
\n### 3.1 لوحة تحكم خدمة العملاء\n- واجهة مخصصة لإدارة طلبات العملاء\n- عرض وإدارة بيانات الخدمات
- متابعة حالة الطلبات

### 3.2 لوحة تحكم العمال
- واجهة مخصصة للعمال لعرض المهام المخصصة لهم فقط
- تحديث حالة المهام
- الوصول المحدود للبيانات الخاصة بكل عامل

### 3.3 نظام الأمان
- تطبيق Row Level Security (RLS) في Supabase لضمان عدم وصول العمال إلا لبياناتهم الخاصة
- استعلامات قاعدة بيانات محسّنة لتحقيق أقصى سرعة أداء
\n### 3.4 إدارة المستخدمين
- نظام تسجيل دخول وصلاحيات متعدد المستويات
- حفظ تفضيلات المستخدم (الثيم) في localStorage
\n## 4. المتطلبات التقنية

### 4.1 حزمة التقنيات
- Frontend: React مع Vite
- Styling: Tailwind CSS
- Backend/Database: Supabase
\n### 4.2 الخطوط\n- الخط الأساسي: Cairo (font-sans)
- خط العناوين: Montserrat (font-heading)
\n### 4.3 الأمان
- تطبيق Row Level Security بصرامة في Supabase
- تحسين جميع استعلامات قاعدة البيانات
\n## 5. أسلوب التصميم

### 5.1 نظام الألوان
- دعم كامل للوضع الداكن والساطع (Dark/Light Mode)
- زر تبديل الثيم في شريط التنقل مع حفظ التفضيل
-ألوان متناسقة مع كلا الوضعين

### 5.2 التفاصيل البصرية
- تأثير الخلفية المتفاعلة مع التمرير (Parallax Effect) باستخدام الكلاس .parallax-bg
- تصميم أنيق ومتسق عبر جميع المكونات
- انتقالات سلسة بين الوضع الداكن والساطع

### 5.3 التخطيط العام
- واجهة نظيفة ومنظمة
- تسلسل منطقي للعناصر
- تجربة مستخدم سلسة ومتكاملة
